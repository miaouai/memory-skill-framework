✨ 核心功能

📦 分层存储

数据模块、情境日志、工作记忆缓存三层分离，结构清晰

💾 持久化

磁盘原子提交，重启不丢失状态，跨会话持续记忆

🔒 安全第一

零硬编码密钥，环境变量管理凭据，通过红队审计

📊 版本友好

纯 JSON 配置，支持 Git 版本控制和增量更新

🏗️ 系统架构

memory/                          # 记忆根目录
├── data/                        # 结构化核心数据 (JSON)
│   ├── apis.json                # API 配置中心
│   ├── profiles.json            # 用户/身份资料  
│   ├── network.json             # 网络/代理设置
│   ├── tasks.json               # 任务模板库
│   ├── error-patterns.json      # 已知错误模式库
│   └── security-guide.json     # 安全协议规范
│
├── episodic/                    # 情境记忆层
│   └── YYYY-MM-DD-xxxx.json    # 按日期分片的事件记录
│
├── working/                     # 工作区临时记忆
│   └── current_session.json    # 活跃会话状态
│
└── index/                       # 快速查找索引表

🚀 快速开始

作为 Copaw 技能安装

copaw skill install https://github.com/miaouai/memory-skill-framework

作为独立模块集成

git clone https://github.com/miaouai/memory-skill-framework.git memory
source memory/scripts/setup_env.sh
